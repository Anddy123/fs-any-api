Fish

This API uses a list of Fish as its resource. The routes are as follows:
Route Method Description Example 
/fishs
 GET Gets a list of fishs' names and ids Example 
/fishs/:id
 GET Returns the detail about the cat matching the id in the parameter Example 
/fishs/:id
 POST Returns the detail about the cat matching the id in the parameter curl -d '{"name": "Steve", "price": 2, "image": "http://placehold.it/500", "description": "new description", "category": "small"}' -H 'Content-Type: application/json' '/api/fishs' 
/fishs
 PATCH Returns the detail about the cat matching the id in the parameter curl -d '{"name": "Steve", "price": 2, "image": "http://placehold.it/500", "description": "updated description", "category": "small"}' -H 'Content-Type: application/json' --request PATCH '/api/fishs/3' 
/fishs
 DELETE Returns the detail about the cat matching the id in the parameter curl --request DELETE 'api/fishs/3'
